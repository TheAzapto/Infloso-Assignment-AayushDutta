# Infloso-Assignment-AayushDutta

Full Stack Developer Assignment Build Login and Signup App with React.js, Node.js and JWT

---

## Objective

---

Node.js API that handles user registration and authentication for a fictional social media platform called "Connectverse". The API should implement secure registration, login, and JWT-based authentication, following best practices.

Develop the login and signup screens for a fictional music streaming service called "MelodyVerse". These screens should be visually appealing, user-friendly, and function flawlessly across different devices (desktop, tablet, mobile).

---

## Important

---
Make sure to have Node.js and npm installed on your system.
to install node.js visit <https://nodejs.org/en/download/>
to install npm visit <https://www.npmjs.com/get-npm>

---

## How to Run the Project

---

First Run `start build.bat` in the root directory to install dependencies and build the project.

Then run `start run.bat` in the root directory to start the project.
